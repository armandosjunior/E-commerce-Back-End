# E-commerce Back End :credit_card:

This application is an e-commerce backend site. An Express.js API was configured for it to use Sequelize and then interact with a MySQL database.

## **Steps to get started:**
```
- npm i (+ dependencies)
- npm run seed
- node server.js
```
